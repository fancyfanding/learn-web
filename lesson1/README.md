## 面试过程

1. 展示项目能直接演示，能够部署最好
2. 项目挖掘，start法则，不要只是描述经验；注意听面试官问题的用意，问了几个问题，是否答全；不可托大，面对不知道的问题可以说不知道，但是要有思考过程
3. 思考时间话术：请您给我十几秒的思考时间
4. 写算法题要先说思路，再去写实际代码
5. 照片换个更利索的
