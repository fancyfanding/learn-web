# 简历优化

## 简历布局和美化

整体左右布局，左侧为基础信息，模块顺序为 contact - education - languages - 兴趣爱好 -skills（ATS关键词）

右上方分别是名字和岗位，下面profile写核心优势，work experience ，荣誉证书

简历篇幅整体不能超过三页

字体格式不能超过两种，一种适用于标题，另一种用于正文，板块间距保持一致，中英文中间空一格，重点单词加粗，技术名词拼写是否有误

## 简历的组成

姓名、岗位、出生日期、地点（想在哪里工作的具体的地址）、电话、github、教育经历、兴趣爱好

**课后作业： github的美化github**

### 2. 核心优势-放到简历顶部

多端一体化的交付能力（小程序、pc、云开发）：具备从用户端到管理端的完整交付经验，包括内容社区、问卷测评、互动闭环等，结合云数据库实现业务闭环

业务组件化和可复用能力：基于vue3生态、将高频的业务能力进行模块化封装，提升迭代的效率，从而降低重复开发的成本

数据分析与可视化：在管理端实现运营的统计和数据分析，使用echars构建关键指标的看板，帮助运营提升决策的效率、管理的效率

性能优化：对于数据密集型场景，和资源加载的策略，有通透的了解，针对pc端表格小程序内容的场景进行分页、按需加载、懒加载、预加载，改善了大数据量下的卡顿与交互的延迟，从而提升流畅度
